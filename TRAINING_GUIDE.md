# AutismCare AI Model Training Guide

This guide explains how to train the LLaMA 3 text model on your ADI.docx data and the ViT image model on your autism face image dataset.

## Overview

The chatbot uses two AI models:

1. **LLaMA 3 (Text Chat)**: Trained on your ADI.docx questionnaire data for answering autism-related questions
2. **Vision Transformer (ViT)**: Trained on your image dataset (autistic/non-autistic children) for image analysis

## Prerequisites

### Python Environment
```bash
# Create virtual environment
python -m venv .venv
```

### For LLaMA 3 Training (Optional - requires GPU)
- NVIDIA GPU with 16GB+ VRAM for full fine-tuning
- Or use Ollama for CPU-based inference (no training)
- Or use Google Colab with free GPU

### For ViT Training
- NVIDIA GPU recommended (4GB+ VRAM)
- CPU training possible but very slow

## Step 1: Train LLaMA 3 on ADI.docx

### Quick Start (Windows PowerShell)

Use the included PowerShell script for easy setup:

```powershell
.\start-chatbot.ps1
# Choose option 1 to train LLaMA 3 RAG model
```

### Option A: RAG-based Approach (Recommended - Fast, No GPU)

This creates a retrieval-based chatbot using sentence transformers:

**Windows (PowerShell):**
```powershell
cd IA-chatboot
python train_llama_adi.py
# Choose option 1 (RAG with Sentence Transformers)
```

**Mac/Linux (Bash):**
```bash
cd IA-chatboot
python train_llama_adi.py
# Choose option 1 (RAG with Sentence Transformers)
```

This will:
1. Load your ADI questions from `adi_questions.json` or `ADI.docx`
2. Create embeddings using `all-MiniLM-L6-v2`
3. Save the model as `llama_adi_rag_model.joblib`

## Step 2: Train ViT on Image Dataset

### Prepare Your Image Data

Organize your images in this structure:
```
image data/
├── Autistic/
│   ├── image1.jpg
│   ├── image2.jpg
│   └── ...
└── Non_Autistic/
    ├── image1.jpg
    ├── image2.jpg
    └── ...
```

### Run Training

```bash
cd IA-chatboot
python train_vit_autism.py
```

This will:
1. Load images from `../image data/` directory
2. Create an ensemble of 3 models (Swin-B, ViT-B/16, ConvNeXt-Tiny)
3. Train for 50 epochs with data augmentation
4. Save the best model as `best_vit_autism_model.pth`
5. Save metrics to `vit_model_metrics.json`

### Training Output

After training, you'll have:
- `best_vit_autism_model.pth` - Trained model weights
- `vit_model_metrics.json` - Training metrics
- `training_curves.png` - Loss/accuracy curves

## Step 3: Start the FastAPI Backend

```bash

```

The server will start on `http://localhost:8000`

### API Endpoints

- `POST /chat` - Chat with LLaMA 3 (ADI data) + GPT-4 fallback
- `POST /predict` - Analyze images with ViT
- `GET /health` - Health check
- `GET /model-metrics` - Model performance metrics

## Troubleshooting

### LLaMA 3 Model Not Loading
- Ensure `llama_adi_rag_model.joblib` exists in `IA-chatboot/`
- Re-run training if needed

### ViT Model Not Loading
- Ensure `best_vit_autism_model.pth` exists in `IA-chatboot/`
- Check that image data directory structure is correct

### Out of Memory (GPU)
- Reduce batch size in training script
- Use gradient accumulation
- Use smaller model (single ViT instead of ensemble)

### Slow Training
- Use GPU if available
- Reduce number of epochs
- Use smaller image size
